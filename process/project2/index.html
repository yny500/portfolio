  • Overview

    군포문화예술회관의 웹 페이지는 단순한 레이아웃의 구조로 되어 있으며, 올드한 디자인을 탈피시켜 접근이 쉽고 보다 편리하게 이용할 수 있도록 리뉴얼 프로젝트를 진행하였습니다.

    Color Identity

    기존 로고 컬러를 사용하여 심플하면서 통일감을 주었습니다.

    • #27313B
    • #A7C6A3
    • #086982
    • #E7E7E7

  • As Is

    공연 정보의 전달이 아쉽고 올드한 디자인을 개선할 필요가 있었으며, 레이아웃의 그리드가 단조로운 문제가 있었습니다.

    To Be

    최근 트렌드인 영상, 이미지 중심의 디자인을 지향하고 있으며, 정보전달이 편안한 내비게이션을 제공하여 사이트를 방문하는 사용자의 효율성을 강조하여 디자인하였습니다.

    • 다양한 정보 제공
    • 편리한 접근의 디자인
