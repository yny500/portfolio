  • Overview

    웹 페이지의 여러 문제점을 분석하여 접근과 이용이 편리한 사이트로 개선할 필요가 있었습니다. 이를 통해 사용자와 커뮤니케이션의 효율성을 높이고 소통하는 브랜드로써의 가치 확립을 위해 리뉴얼 프로젝트를 진행하였습니다.

    Color Identity

    로고를 제작해 이를 바탕으로 아이덴티티 컬러를 지정하여 통일감을 주었습니다.

    • #212121
    • #F89A1B
    • #FE1D25
    • #200333

  • As Is

    플래시 콘텐츠 종료로 인해 웹 표준에 맞게 제작함과 동시에 올드한 디자인을 개선할 필요가 있었습니다.

    To Be

    사이트는 편리한 접근성을 위한 디자인을 지향하고 있으며, 노을바다펜션과의 통일감, 일관성을 통한 정보전달이 편안한 내비게이션을 제공하여 사이트와 사용자간의 커뮤니케이션의 효율성을 강조하여 디자인하였습니다.

    • 플래시 콘텐츠 종료
    • 올드한 디자인
    • 웹 표준에 맞춰 제작
    • 편리한 접근의 디자인
